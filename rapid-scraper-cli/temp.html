rapid-scraper-cli

cli tool for web scraping. you can get links,images,description,title,keywords,social_media links etc of a website using commands.

Installation:

  • npm install rapid-scraper-cli

command:

    get <info> <website_url>

commands available:

  • get links url
  • get description url
  • get images url
  • get social url
  • get para url
  • get title url
  • get keywords url
